Key Features

Admin / Organizer Dashboard
Add/edit/delete events, upload banners, track registrations, manage cancellations, and confirm payments.
Payments (M-Pesa)
Store transaction references, amounts, statuses, and generate receipts/tickets.
Virtual Events
Manage online links and track virtual attendance for hybrid participation.
User Dashboard
Registered events card layout, payment history, reminders/notifications, virtual links, and level points.
Feedback & Ratings
Collect post-event reviews and ratings to improve future events.
Modern UI
Dark/purple event theme with responsive layout (CSS/JS) and Django templates.

Run the Django App Locally

GitHub Pages is static; Django needs a server.
Commands (Windows PowerShell)
1) Create/activate venv (already created in this repo): .\.venv\Scripts\Activate.ps1
2) Install deps: python -m pip install -r requirements.txt (optional if already installed)
3) Migrate: python manage.py migrate
4) Create admin: python manage.py createsuperuser
5) Run: python manage.py runserver
Then open: http://127.0.0.1:8000/
